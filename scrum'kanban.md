## 2. Descripción del Proyecto
Este proyecto consiste en una aplicación web funcional orientada a una tienda de prestamo y fletes de ropas folkloricas para las entradas y distitos tipos de danzas para que la gente se vea segura de lo que use y verse elegante ya que es necesario para que tenga un sitio web que para cuando la gente vea y tenga la libertada de escogerby conocer los precios y los detalles de cada vestimenta.
## 3. Estructura Obligatoria del Repositorio
El proyecto ha sido reestructurado rigurosamente bajo el estándar solicitado para el segundo parcial:
* `index.html`: Página principal de la aplicación.
* `css/styles.css`: Hoja de estilos unificada del sitio.
* `js/main.js`: Lógica y dinamismo con JavaScript.
* `assets/`: Almacenamiento de imágenes y recursos iconográficos del catálogo.
* `docs/`: Documentación formal del proyecto (Investigación, manuales).
* `scrum-kanban/`: Planificación y control de actividades del desarrollo.

## 4. Requisitos de Ejecución
Para visualizar el proyecto localmente:
1. Clonar este repositorio público.
2. Abrir el archivo `index.html` en cualquier navegador web moderno (Chrome, Edge, Firefox).
